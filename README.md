# 🚀 AutoRoAppeal - Fully Automatic Roblox Appeal AI

**An open-source, free, and fully automatic alternative to roappeal.com**

AI-powered tool that instantly drafts professional, tailored Roblox ban appeals. Review, edit, and submit yourself. Handle follow-up replies automatically too.

> ⚠️ **Important Disclaimer**: This is a community tool to help you write better appeals. It does **not** guarantee unbans. Roblox moderation decisions are final in most cases. We are **not affiliated** with Roblox Corporation. Always be honest in your appeals. Submitting false information can worsen your situation.

## Features

- ✅ **Fully Automatic AI Drafting** — Describe your ban once, get a polished appeal in seconds
- ✅ **Smart Follow-ups** — Paste Roblox's reply and generate the next response automatically
- ✅ **Privacy First** — Everything runs in your browser. No data sent to servers
- ✅ **Free Forever** — No paywalls, no subscriptions, no hidden fees
- ✅ **Customizable** — Choose tone, edit the output, regenerate
- ✅ **Educational** — Learn best practices for appeals
- ✅ **Open Source** — Transparent, auditable, community improvable

## Live Demo

## How to Use

1. Go to the Demo section on the website
2. Select your ban type (Enforcement, Moderation, etc.)
3. Paste your ban notification text or describe the situation in detail
4. Add any extra context or what you want to emphasize
5. Choose preferred tone
6. Click **Generate Appeal Automatically**
7. Review the output, make edits if needed, then copy it
8. Submit through Roblox's official Help Center appeal process
9. For ongoing conversations: Use the **Follow-up Replies** tab — paste their response and generate the next message

## Tech Stack

- Single-file modern website (index.html)
- Tailwind CSS (via CDN)
- Vanilla JavaScript for fully automatic generation
- 100% client-side → maximum privacy and simplicity

Future: Can easily be extended to a full Next.js app with real LLM backend.
